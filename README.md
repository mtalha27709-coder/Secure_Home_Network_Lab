# Secure Home Network Lab

## Objective
Configure a wired and wireless home network using Cisco Packet Tracer, including DHCP, IP addressing, and WPA2 security.

## Skills Learned
- IP Addressing & DHCP Configuration
- Wired & Wireless Network Setup
- Router Configuration & Management
- Wireless Security (WPA2)
- Network Connectivity Testing
- Troubleshooting & Problem Solving
- Practical Network Administration
- Packet Tracer Simulation

## Tools Used
- Cisco Packet Tracer
